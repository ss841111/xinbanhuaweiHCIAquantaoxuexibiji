1. **HCIA·第一堂课 ENSP与初探网络.pdf** - 推开网络世界的大门，从模拟环境ENSP入手。

4. **HCIA·第九堂课 TCP·UDP及路由表概述.pdf** - 网络传输的两大支柱与路径选择的基础知识。
5. **HCIA·第二堂课 网络的基本概念(物理层·数据链路层).pdf** - 从比特流到帧，了解底层通讯原理。
